What is Debian Live Studio?

Debian Live Studio is a web-based tool for building customised Debian images that do not require installation before they can be used. Live images can can book from various types of media, including CDs, USB sticks or via netbooting.

How do I report bugs?

Please email your bug report to debian-live@lists.debian.org or report it on the #debian-live IRC channel on OFTC.

How can I hack on Debian Live Studio?

You can download the source using git:

  $ git clone git://live.debian.net/git/live-studio.git

You can then either send a merge request or a patch series to debian-live@lists.debian.org. It is recommended that you join the #debian-live IRC channel on OFTC to co-ordinate larger changes.
